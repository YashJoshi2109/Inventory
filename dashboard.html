⚙ RFID Integration Notes

This dashboard will integrate with your physical RFID system via the Python backend.


# Python RFID webhook endpoint (in app.py):
@app.route('/api/rfid/<tag_id>')
def rfid_scan(tag_id):
  result = inventory_manager.rfid_lookup(tag_id)
  return jsonify(result)

When an RFID reader scans a tag, it sends the tag ID to this endpoint. The system looks up the item, shows details, and can auto-log transactions.

// python api
Command-line interface and API reference
🐍 inventory_manager.py — Command Reference
Run from the directory containing lab_inventory_v2.xlsx
QUERY INVENTORY
# Show all items
python inventory_manager.py query

# Filter by SKU
python inventory_manager.py query --sku LAB-001

# Show all Reagents
python inventory_manager.py query --category Reagents

# Show items needing reorder
python inventory_manager.py query --status "REORDER NOW"

# Find by RFID tag
python inventory_manager.py query --rfid RFID-0001

# Export as JSON
python inventory_manager.py query --json
RECEIVE STOCK
python inventory_manager.py receive \
  --sku LAB-001 \
  --qty 500 \
  --supplier "Fisher Scientific" \
  --po PO-2024-001 \
  --received-by "Dr. Smith"
LOG USAGE
python inventory_manager.py use \
  --sku LAB-002 \
  --qty 200 \
  --type "Experiment Use" \
  --project EXP-2024-003 \
  --issued-by "Lab Tech A"
ADD NEW ITEM
python inventory_manager.py add-item \
  --sku LAB-011 \
  --name "Acetone HPLC Grade" \
  --category Reagents \
  --location Flammables \
  --unit mL \
  --cost 0.09 \
  --qty 0 \
  --reorder-level 200 \
  --supplier "Sigma-Aldrich"
REORDER REPORT & SYNC
# Print reorder report
python inventory_manager.py reorder

# Export reorder list to CSV
python inventory_manager.py reorder --export

# Sync and validate database
python inventory_manager.py sync

# API endpoints:
GET /api/inventory # All items
GET /api/inventory/<sku> # Single item
POST /api/receive # Log receipt
POST /api/use # Log usage
POST /api/add-item # Add new SKU
GET /api/reorder # Reorder list
GET /api/rfid/<tag_id> # RFID lookup
